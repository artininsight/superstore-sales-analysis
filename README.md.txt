## Dataset
- **Source**: [Kaggle - Superstore Sales](https://www.kaggle.com/datasets/bhanupratapbiswas/superstore-sales)
- **Records**: 9,994 orders
- **Key Columns**: `Sales`, `Profit`, `Quantity`, `Discount`, `Order Date`, `Category`
